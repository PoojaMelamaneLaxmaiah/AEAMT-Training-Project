# AEAMT-Training-Project
The website was developed to represent and support the mission of the Academy of Excellence for Advanced Manufacturing Technology (AEAMT), an initiative by the Central Manufacturing Technology Institute (CMTI) aimed at addressing the critical need for industry-ready engineers in India's manufacturing sector.

## Features
- Student/trainee registration
- Project submission and management
- Dynamic content management system
- Admin panel for training supervisors
- Database with efficient queries

## Tech Stack
- PHP
- MySQL
- HTML5, CSS3, JavaScript
- Bootstrap (for UI)

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/poojamelamanelaxmaiah/AEAMT_Training_Project.git
2. Import the MySQL database.
3. Configure connection settings in config.php.
4. Run in local server environment (XAMPP/WAMP).
